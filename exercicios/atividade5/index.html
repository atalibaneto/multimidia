HyperText Markup Language - HTML

HTML, sigla para "HyperText Markup Language", é uma linguagem de marcação usada para estruturar conteúdo em páginas da web, definindo elementos como parágrafos, cabeçalhos, listas, imagens e outros componentes. Apesar de não ser uma linguagem de programação no sentido estrito, é fundamental para a criação e organização do conteúdo visível em um navegador.

Função do HTML:

    Estrutura e conteúdo:
  • Define a estrutura básica de uma página web, indicando como o conteúdo deve ser organizado (ex: parágrafos, títulos, listas).
    • Conteúdo semântico:
  • Permite adicionar significado ao conteúdo, indicando se um elemento é um título, um parágrafo, uma imagem, etc.
    • Links e hipertextos:
  • Permite criar links que conectam diferentes páginas da web e facilita a navegação.
    • Base da Web:
  • É a base fundamental para a criação de páginas web, juntamente com outras tecnologias como CSS e JavaScript.

O que o HTML NÃO é:

    Linguagem de programação:
  • O HTML não permite criar lógica dinâmica ou funcionalidades complexas como outras linguagens de programação.
    • Estilização:
  • Embora o HTML possa ser usado para estilizar elementos básicos (ex: cor de texto), a estilização mais avançada é feita com CSS.
    • Interação:
  • A interação do usuário com a página (ex: eventos, botões, etc.) é geralmente implementada com JavaScript.
